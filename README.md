# UB-Institutional-Profile-generator-and-UBIP-
An interactive HTML/JavaScript tool to generate the University of Bridgeport Institutional Profile (UBIP).
University of Bridgeport Institutional Profile (UBIP) Generator

## Project Overview
This project is an **interactive, front-end generator** designed to streamline the creation of the annual University of Bridgeport Institutional Profile (UBIP). Developed as an efficiency solution for the Office of Institutional Effectiveness (OIE), it allows users to input various data points—such as enrollment figures, financial metrics, and faculty statistics—and instantly generate a standardized, presentable HTML document for review or submission.

## Key Features
* **Data Input Form:** A structured form for entering all required institutional data points.
* **Instant Profile Generation:** Converts input data into a clean, formatted HTML document in real-time.
* **Easy Download:** Includes a function to download the final profile as a standalone HTML file for archiving or distribution.
* **Client-Side Operation:** Built purely with **HTML, CSS, and JavaScript**; no server-side dependencies.

## Technologies Used
* HTML5
* CSS3 (Styling and Layout)
* JavaScript (Data handling and DOM manipulation)

## How to View and Use
1.  **Clone the Repository** or download the `University of Bridegport Intitutional Profile genarator - 2025.html` file.
2.  **Open the HTML file** in any modern web browser (Chrome, Firefox, etc.).
3.  **Input the required data** into the form fields.
4.  Click the **"Generate Institutional Profile"** button to view the results.
5.  Click **"Download Profile"** to save the final report.
